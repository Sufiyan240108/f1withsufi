# Security Policy

## Supported Versions

Only the latest version of **f1withsufi** is currently supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| v1.x    | :white_check_mark: |
| < v1.0  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please do not open a public issue. Instead, please report it via one of the following methods:

- **GitHub Vulnerability Reporting:** Use the "Report a vulnerability" button on the Security tab.
- **Contact:** Reach out to the maintainer directly if contact info is available in the profile.

We will acknowledge your report and provide an estimated timeline for a fix.
